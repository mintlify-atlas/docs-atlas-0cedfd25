## What is Vicenta?

Vicenta is a modern web application built with Next.js 14, React 18, and TypeScript. It provides a solid foundation for building scalable, type-safe web applications with excellent developer experience and performance out of the box.

## Technology Stack

Vicenta is built with a modern, carefully selected technology stack:

- **Framework**: Next.js 14.2.6 with App Router
- **UI Library**: React 18 with React DOM
- **Language**: TypeScript 5 with strict mode enabled
- **Styling**: Tailwind CSS 3.4.1 with custom gradient utilities
- **Code Quality**: ESLint with Next.js configuration
- **Font Optimization**: Next.js font optimization with Inter font family

## Project Structure

<Note>
  Vicenta follows the Next.js 14 App Router convention with the `src/app` directory structure.
</Note>

The application is organized as follows:

```bash
front/
├── src/
│   └── app/
│       ├── layout.tsx      # Root layout with metadata
│       ├── page.tsx        # Home page component
│       ├── globals.css     # Global styles with Tailwind directives
│       └── favicon.ico     # Application favicon
├── public/                 # Static assets
├── tailwind.config.ts      # Tailwind configuration
├── next.config.mjs         # Next.js configuration
├── tsconfig.json          # TypeScript configuration
└── package.json           # Dependencies and scripts
```

## TypeScript Configuration

Vicenta is configured with strict TypeScript settings for maximum type safety:

```json package.json
{
  "compilerOptions": {
    "strict": true,
    "moduleResolution": "bundler",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

<Tip>
  The `@/*` path alias allows you to import modules from the `src` directory without relative paths.
</Tip>

## Custom Tailwind Configuration

Vicenta extends Tailwind CSS with custom gradient utilities:

```typescript tailwind.config.ts
theme: {
  extend: {
    backgroundImage: {
      "gradient-radial": "radial-gradient(var(--tw-gradient-stops))",
      "gradient-conic":
        "conic-gradient(from 180deg at 50% 50%, var(--tw-gradient-stops))",
    },
  },
}
```

These custom gradients enable sophisticated visual effects used throughout the application.
